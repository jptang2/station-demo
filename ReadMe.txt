station-demo

Name:JianPing Tang


This is a rendered scene about a train station.
I design the scene in 3Ds Max and I use DXUT to build a program. Although there is a lot of texture mapping bug, but I still working on it.
-------------------------------------------------
Bin folder contains executable files.
Include folder contains DXUT files.
Lib folder contains DXUT static library.
media folder contains resources files.
source folder contains source code.

this project compiled by vs2012, it requires win7 or higher to run.

Finished in 2014.12
